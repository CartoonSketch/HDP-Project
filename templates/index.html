About the Project

This project leverages advanced Machine Learning (TabPFN Transformer) models to analyze multiple health factors such as blood pressure, cholesterol, diabetes, and lifestyle habits. The system provides a detailed report on the probability of heart disease risk, along with recommendations for healthier living.
